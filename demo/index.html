Demo: lcwllmr/vi-mode.js ()

This is a live demo for github:lcwllmr/vi-mode.js, a TypeScript library providing a lightweight HTML text editor component with vi-like keybindings. Check out GitHub for current state of development and documentation. See here for the test coverage report.

Click on the editor area below to focus it, then try editing. Of course, this is intended to be run with a physical keyboard.
